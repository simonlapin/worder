# Политика конфиденциальности Worder / Worder Privacy Policy

Обновлено: 2 июля 2026 · Updated: July 2, 2026

## Русский

Worder — офлайн-приложение для заучивания английских слов.

- Все данные (словарь, прогресс обучения, история занятий, настройки) хранятся только на вашем устройстве. У приложения нет серверов, аккаунтов и аналитики.
- Приложение не собирает, не передаёт и не продаёт никакие персональные данные.
- Опционально вы можете указать собственный API-ключ Anthropic. В этом случае приложение отправляет в Anthropic API только изучаемые английские слова — чтобы сгенерировать предложения-примеры и подсказки. Ключ хранится в Keychain устройства, никуда не экспортируется и может быть удалён в настройках в любой момент. Без ключа приложение полностью функционально.
- Обработка данных Anthropic описана в их политике: https://www.anthropic.com/legal/privacy
- Резервная копия создаётся только по вашей команде и сохраняется туда, куда вы её экспортируете. API-ключ в неё не включается.

Вопросы: simnlapn@gmail.com

## English

Worder is an offline English vocabulary learning app.

- All data (dictionary, learning progress, session history, settings) is stored only on your device. The app has no servers, no accounts, and no analytics.
- The app does not collect, transmit, or sell any personal data.
- Optionally, you may provide your own Anthropic API key. In that case the app sends only the English words you study to the Anthropic API — to generate example sentences and hints. The key is stored in the device Keychain, is never exported, and can be deleted in Settings at any time. The app is fully functional without a key.
- Anthropic's data handling is described in their policy: https://www.anthropic.com/legal/privacy
- Backups are created only on your command and saved wherever you export them. The API key is never included.
