# Worder — английские слова

Офлайн-тренажёр для заучивания 1500 самых употребляемых английских слов с умными повторениями (FSRS): упражнения в обе стороны (EN → RU и RU → EN), свободная тренировка, статистика, напоминания и резервное копирование. Все данные хранятся только на вашем устройстве.

Worder is an offline vocabulary trainer for the 1500 most common English words with FSRS spaced repetition. All data stays on your device.
