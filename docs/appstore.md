## Описание (RU)

Worder помогает выучить 1500 самых употребляемых английских слов до уверенного перевода в обе стороны — EN → RU и RU → EN.

Приложение само управляет процессом:

• Умные повторения по алгоритму FSRS — слова показываются на грани забывания, когда повтор эффективнее всего.
• Тип упражнения растёт вместе со словом: знакомство → выбор из четырёх → ввод перевода → аудирование и подстановка слова в предложение.
• Раздельный учёт направлений: слово считается выученным, только когда вы уверенно переводите его в обе стороны.
• Свободная тренировка без ограничений: весь словарь в любой момент, трудные слова выпадают чаще.
• Трудные слова отслеживаются автоматически и получают особое внимание.
• Статистика: прогресс, история занятий, стрик, браузер всех слов с поиском и сортировками.
• Напоминания о занятиях с числом слов к повторению.
• Резервная копия всего прогресса в один файл и восстановление из него.

Worder работает полностью офлайн. Опционально можно подключить собственный API-ключ Anthropic — тогда приложение сгенерирует предложения-примеры и подсказки для трудных слов; без ключа доступно всё остальное.

Без аккаунтов, рекламы и сбора данных: весь прогресс хранится только на вашем устройстве.

## Анкета приватности (App Privacy)

- **Data Not Collected** — приложение не собирает никаких данных.
- Обоснование: серверов нет; всё хранится локально (SwiftData/Keychain/UserDefaults). Пользовательский Anthropic-ключ — опционален, вводится пользователем добровольно, запросы уходят напрямую в Anthropic с устройства, приложение-посредник данные не собирает и не хранит вне устройства.
